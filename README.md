# 🥗 NutriSmart AI  
### AI-Powered Smart Food Choice & Healthy Habit Web App

> Helping people make better food decisions through AI, behavioral insights, and contextual recommendations.

---

# 📌 Overview

NutriSmart AI is a smart health-tech web application designed to help individuals make healthier food choices and build long-term eating habits using:

- Nutrition data  
- User behavior  
- Context-aware recommendations  
- Habit-building systems  
- AI-powered food intelligence

Unlike traditional calorie trackers, NutriSmart acts as a **Food Decision Assistant**, guiding users toward smarter choices in everyday life.

**Tagline:**  
**“Waze for Food Decisions.”**

---

# 🚨 Problem Statement

Millions struggle with:

- Poor food choices
- Inconsistent eating habits
- Nutrition confusion
- Lack of personalized guidance
- Balancing health, budget, and convenience

Most food apps track calories.

Few help users make intelligent decisions.

NutriSmart solves this gap.

---

# 💡 Solution

NutriSmart AI combines:

- Smart nutrition recommendations
- Food scoring intelligence
- Context-aware suggestions
- Habit formation tools
- Behavioral insights

to help users improve nutrition decisions daily.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/nutrismart-ai.git
cd nutrismart-ai
```

---

## Create Virtual Environment

```bash
python -m venv venv
```

Activate:

### Windows
```bash
venv\Scripts\activate
```

### Linux / Mac
```bash
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Application

```bash
python app.py
```

App runs at:

```bash
http://127.0.0.1:5000/
```
